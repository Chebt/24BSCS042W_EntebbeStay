# Project Title: EntebbeStay Hotel Booking System

## System Description
EntebbeStay is a dynamic web application designed to help travelers find and book 2-5 star
 hotels located between Kawuku and Kitoro in Entebbe. The system allows users to view hotel 
 details,  services offered, and real-time availability.

## Technologies Used
- **Frontend:** HTML5, CSS3 (External Stylesheet)
- **Backend:** PHP 8.x
- **Database:** MySQL
- **Environment:** VS Code & Docker

## How to Run the Project
1. Clone this repository to your local machine.
2. If using Docker, run `docker-compose up -d`. If using XAMPP, place the folder in `htdocs`.
3. Import the `database.sql` file into your MySQL database (phpMyAdmin).
4. Update `db_config.php` with your database credentials.
5. Open `http://localhost/24BSCS042W_EntebbeStay` in your browser.

## Database Import Instructions
1. Open phpMyAdmin.
2. Create a new database named `entebbe_hotels`.
3. Click on the **Import** tab.
4. Choose the `database.sql` file from the project directory.
5. Click **Go**.
